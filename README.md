# terraform-s3-module

A simple terraform s3 module.

## Pre-requisites

- [Terraform](https://www.terraform.io/)
- [AWS CLI](https://aws.amazon.com/cli/)


## Usage

Look at the [examples](https://github.com/jdxlabs/terraform-s3-module/tree/main/examples) directory.
